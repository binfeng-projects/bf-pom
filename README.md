# 关于本工程
1. 所有binfeng-projects维护的工程的parent父pom由本工程维护

2. spring-boot-dependencies会依赖spring-framework-bom

3. 本工程是纯pom工程,binfeng-projects维护工程默认打SNAPSHOT，一来方便覆盖，二来防止改造的三方工程和官方版本冲突
